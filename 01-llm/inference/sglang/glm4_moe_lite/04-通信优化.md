### 2.2 should_allreduce_fusion 标志

```python
# 由 LayerCommunicator 在 forward 中传递
# 当 should_allreduce_fusion=True 时:
# 1. 跳过 down_proj 的 All-Reduce
# 2. 由后续 input_layernorm 融合处理
```

### 2.3 融合 RMSNorm + All-Reduce

```python
# layers/layernorm.py:300-327
def forward_with_allreduce_fusion(
    self,
    x: torch.Tensor,
    residual: Optional[torch.Tensor] = None,
):
    if residual is not None:
        from sglang.srt.layers.flashinfer_comm_fusion import (
            flashinfer_allreduce_residual_rmsnorm,
        )

        if get_tensor_model_parallel_world_size() > 1:
            # 融合 All-Reduce + Residual + RMSNorm
            fused_result = flashinfer_allreduce_residual_rmsnorm(
                input_tensor=x,
                residual=residual,
                weight=self.weight,
                eps=self.variance_epsilon,
            )
            if fused_result[0] is not None:
                return fused_result

    return self.forward(x, residual)
```

## 3. Reduce-Scatter 优化

### 3.1 allow_reduce_scatter 配置

```python
# glm4_moe_lite.py:410
self.layer_communicator = LayerCommunicator(
    ...
    allow_reduce_scatter=True,  # 启用 Reduce-Scatter
    ...
)
```

### 3.2 Reduce-Scatter vs All-Reduce

```
All-Reduce:
每个 GPU 获得完整的求和结果

Reduce-Scatter:
每个 GPU 只获得部分结果
减少通信量，适合 DP Attention 场景
```
