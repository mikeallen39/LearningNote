## 1. BumpAllocator 内存池

### 1.1 源码位置

`utils/common.py` (LazyValue 和 BumpAllocator 定义)

### 1.2 类定义

```python
# utils/common.py
class BumpAllocator:
    """线性分配内存池，O(1) 分配"""

    def __init__(self, size: int, dtype: torch.dtype, device: torch.device):
        self.buffer = torch.empty(size, dtype=dtype, device=device)
        self.offset = 0
        self.size = size

    def allocate(self, num_elements: int) -> torch.Tensor:
        """分配指定数量的元素，O(1) 时间"""
        if self.offset + num_elements > self.size:
            raise MemoryError("Allocator out of memory")
        start = self.offset
        self.offset += num_elements
        return self.buffer[start:self.offset]

    def reset(self):
        """重置分配器，O(1) 时间"""
        self.offset = 0

    def free(self):
        """释放所有内存"""
        self.offset = 0
```

### 1.3 在 Glm4MoeLiteMLP 中的使用

```python
# glm4_moe_lite.py:122-157
def forward(
    self,
    x,
    forward_batch=None,
    should_allreduce_fusion: bool = False,
    use_reduce_scatter: bool = False,
    gemm_output_zero_allocator: BumpAllocator = None,  # 内存池参数
):
    # 空输入处理
    if (self.tp_size == 1) and x.shape[0] == 0:
        return x

    # 量化权重处理
    if not hasattr(self.gate_up_proj, "weight"):
        self.gate_up_proj.weight = getattr(self.gate_up_proj, "weight_packed")
    if not hasattr(self.down_proj, "weight"):
        self.down_proj.weight = getattr(self.down_proj, "weight_packed")

    # 使用内存池预分配 GEMM 输出
    if (
        gemm_output_zero_allocator is not None
        and x.shape[0] <= 256                          # 小批量
        and self.gate_up_proj.weight.dtype == torch.uint8  # 量化权重
    ):
        # 从内存池分配
        y = gemm_output_zero_allocator.allocate(
            x.shape[0] * self.gate_up_proj.output_size_per_partition
        ).view(x.shape[0], self.gate_up_proj.output_size_per_partition)
        x = (x, None, y)  # 传递给量化 GEMM kernel

    gate_up, _ = self.gate_up_proj(x)
    x = self.act_fn(gate_up)
    x, _ = self.down_proj(
        x,
        skip_all_reduce=should_allreduce_fusion or use_reduce_scatter,
    )
    return x
```

### 1.4 内存池优势

| 操作 | 传统分配 | BumpAllocator |
|------|----------|---------------|
| 分配 | `torch.empty()` 可能触发 cudaMalloc | O(1) 指针移动 |
| 释放 | 等待 GC | `reset()` O(1) |
| 碎片 | 严重 | 无 |

# utils/common.py
class LazyValue:
    """延迟计算，只在第一次访问时执行"""

    def __init__(self, fn: Callable):
        self.fn = fn
        self._value = None
        self._computed = False

    def __call__(self):
        if not self._computed:
            self._value = self.fn()
            self._computed = True
        return self._value
```

## 3. PPMissingLayer 节省内存

### 3.1 源码位置

### 3.2 类定义

# layers/utils.py
class PPMissingLayer(nn.Module):
    """Pipeline Parallel 中未使用的层的占位符"""

    def __init__(self, return_tuple=False):
        super().__init__()
        self.return_tuple = return_tuple

    def forward(self, *args, **kwargs):
        if self.return_tuple:
            return (None,)
        return None
```
