摆脱翻译软件的限制

学会快速找到论文中的关键部分：

- motivation（论文提出的动机）
- method（如何解决该问题）
- effect（提出方法的效果如何）

准备一个本子（或者一个电子文档）在阅读论文的时候记录下思考or值得进一步研究的方向用于后续与老师、同学交流探讨。

对于自己准备深入研究的领域建立一个数据库来记录自己阅读过的文献

如何筛选或者确定一篇论文是否有必要读：

- 看标题、作者
- 看摘要
- 看结论

阅读文献的原则：精读不必求全，略读保有底线。

读读 Intro 时不必一字一句读，着重关注不足和本论文的工作即可，也就是有转折词 however 等的地方和有 we/our 的地方。



# 李沐读论文

3遍法：

- 第一遍只看标题、摘要、结论，简单看一下文章中出现的图和表。第一遍读完后决定这篇论文是否有必要继续阅读、是否对自己有帮助
- 第二遍粗略地从头读到尾，把握文章整体上解决什么问题、用了什么方法即可
- 第三遍精读，边读边思考，逐字逐句弄懂
